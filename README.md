Main Feature
-------------
* Show github user search result with SwipeableCard 
* Load the next page of search result if the SwipeableCard reach its threshold

Tech Stack
----------
* Jetpack Compose
* Coroutine + Flow
* Retrofit
* MVVM
* Landscapist Coil
* Hilt for DI
* Paging3
* Modularization

Environment
-----------
Build with:
- Target SDK = 33
- Build Tools 30.0.3
- AGP 7.4
